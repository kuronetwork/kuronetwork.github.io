## 💡 使用建議

### 日常維護
```typescript
// 新增文章只需編輯 data/articles.ts
{
  id: 60,
  title: "新文章標題",
  date: "2024-12-14",
  category: "資訊安全",
  tags: ["標籤"],
  url: "https://medium.com/...",
  featured: false
},
```

### 更新個人資訊
- 編輯 `app/about/page.tsx`
- 編輯 `app/page.tsx` (首頁)

### 修改樣式
- 編輯 `tailwind.config.ts` (配色)
- 編輯 `app/globals.css` (全域樣式)
